Cara menambahkan suara sorak sendiri
Letakkan file audio cheer.mp3 di folder yang sama dengan file HTML ini, lalu tambahkan baris ini sebelum tag </body>:

<audio id="cheer-audio" src="cheer.mp3" preload="auto"></audio>

File sudah otomatis mencari cheer.mp3 — kalau ada, akan diputar saat pop-up muncul. Tombol 🔊 di pojok kanan bawah untuk on/off suara.
